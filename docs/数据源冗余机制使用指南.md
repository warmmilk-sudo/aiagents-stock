# 数据源机制使用指南

## 功能概述

当前系统只使用 Tushare 作为 A 股外部数据源。A 股历史行情、基础信息、资金流等数据以 Tushare 为主；实时行情优先使用 TDX，本地 TDX 不可用时再按模块配置回退到 Tushare 或直接返回错误。

## 配置步骤

1. 访问 [Tushare 官网](https://tushare.pro) 注册并获取 Token。
2. 在系统的环境配置页面填写 `TUSHARE_TOKEN`，或在 `.env` 中配置：

```bash
TUSHARE_TOKEN="your_tushare_token_here"
TUSHARE_URL="https://api.tushare.pro"
```

3. 重启后端服务，让新的数据源配置生效。

## 数据获取流程

```text
深度分析 / A 股历史行情
  -> Tushare daily / fund_daily
  -> 成功：返回标准 OHLCV 数据
  -> 失败：返回明确错误原因，停止分析

实时行情
  -> TDX 本地服务
  -> Tushare 或模块自身兜底
  -> 失败：返回错误或空行情
```

## 常见问题

### 提示 token expired

说明当前 `TUSHARE_TOKEN` 已过期或对应的代理服务拒绝访问。更新 Token 后重启服务。

### 提示未返回历史数据

检查股票代码、Tushare 权限、交易日范围和 `TUSHARE_URL`。深度分析不会切换到其它数据源。

### TDX 行情失败

检查 `TDX_CONFIG` 或 `TDX_BASE_URL` 指向的本地服务是否可用；TDX 只负责实时行情，不替代深度分析所需的历史日线。
