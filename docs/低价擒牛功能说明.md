# 低价擒牛功能说明

## 功能定位

低价擒牛是一个面向低价高成长标的的选股板块，帮助你快速筛选候选股票，并提供策略模拟结果辅助判断。

## 当前能力

- 低价股候选池筛选
- 基于规则的策略打分
- 策略结果表格与图表展示
- 与主系统一致的“提醒 + 人工确认”工作流

## 执行语义

低价擒牛板块只提供：

- 选股结果
- 策略模拟
- 后续加入观察或持仓管理的参考

它不会直接接入券商交易，也不会自动下单。

## 推荐使用方式

1. 运行筛选得到候选股票。
2. 查看策略评分、风险提示和关键指标。
3. 将感兴趣的标的加入观察或资产账本。
4. 结合智能盯盘和价格预警继续跟踪。

## 与主系统的关系

- 低价擒牛负责“发现机会”。
- 智能盯盘负责“盘中分析”。
- 价格预警负责“价位提醒”。
- 最终成交仍由用户手工完成。

## 常见问题

### 策略模拟是否等于实盘收益？

不是。策略模拟只用于验证规则逻辑，不代表实际成交结果。

### 能否直接从这里自动下单？

不能。系统当前不提供自动交易能力。
