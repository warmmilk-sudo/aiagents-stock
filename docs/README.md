## 当前执行模式

- 系统已统一为“分析 + 提醒 + 手工登记”模式。
- 智能盯盘会产出 `BUY / SELL / HOLD` 建议，并为买卖建议生成待处理动作。
- 价格预警只负责监听实时价格并触发进场、止盈、止损提醒。
- 系统不再提供券商自动下单相关能力。

## 推荐阅读顺序

1. 先看 [环境配置快速指南.md](环境配置快速指南.md) 完成最小配置。
2. 再看 [智能盯盘配置说明.md](智能盯盘配置说明.md) 确认盯盘相关参数。
3. 使用时参考 [智能盯盘使用指南.md](智能盯盘使用指南.md) 处理 AI 建议和待办。
