# 环境配置功能说明

## 配置页结构

主程序配置页目前包含三个标签：

1. 基本配置
2. 数据源配置
3. 通知配置

系统已不再提供量化交易配置页。

## 基本配置

### DeepSeek

- `DEEPSEEK_API_KEY`：DeepSeek API Key
- `DEEPSEEK_BASE_URL`：DeepSeek 接口地址
- `DEEPSEEK_MODEL`：默认分析模型

### 应用运行

- `STREAMLIT_SERVER_PORT`：Streamlit 端口
- `STREAMLIT_SERVER_ADDRESS`：Streamlit 监听地址

## 数据源配置

### TDX

- `TDX_ENABLED`：是否启用 TDX
- `TDX_BASE_URL`：TDX HTTP 服务地址

说明：

- 启用后，实时监测优先读取 TDX 报价。
- 不启用时自动回退默认数据源。

## 通知配置

### 邮件

- `EMAIL_ENABLED`
- `SMTP_SERVER`
- `SMTP_PORT`
- `EMAIL_FROM`
- `EMAIL_PASSWORD`
- `EMAIL_TO`

### Webhook

- `WEBHOOK_ENABLED`
- `WEBHOOK_TYPE`
- `WEBHOOK_URL`
- `WEBHOOK_KEYWORD`

## `.env` 示例

```env
DEEPSEEK_API_KEY="your_api_key"
DEEPSEEK_MODEL="deepseek-chat"

TDX_ENABLED="false"
TDX_BASE_URL="http://127.0.0.1:8181"

EMAIL_ENABLED="false"
WEBHOOK_ENABLED="false"
```

## 当前执行语义

- 智能盯盘：生成 AI 建议和待处理动作。
- 价格预警：生成进场、止盈、止损提醒。
- 手工登记：成交后由用户回填到系统。

系统不再连接券商自动交易接口，也不会直接下单。
