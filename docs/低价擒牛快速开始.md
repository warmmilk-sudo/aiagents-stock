# 低价擒牛快速开始

## 1. 启动应用

```bash
streamlit run app.py --server.port 8503 --server.address 127.0.0.1
```

## 2. 进入低价擒牛页面

在左侧菜单进入选股板块，打开“低价擒牛”。

## 3. 执行筛选

- 设定筛选范围
- 运行策略
- 查看候选股票列表和策略评分

## 4. 处理结果

- 将重点标的加入观察列表
- 对已持仓标的结合智能盯盘继续跟踪
- 用价格预警管理关键价位

## 5. 记住边界

- 这里的策略模拟只用于研究参考。
- 系统不会从低价擒牛页面自动下单。
- 如需成交，请在外部交易终端完成，再回到系统手工登记。
