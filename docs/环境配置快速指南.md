# 环境配置快速指南

## 最小启动配置

```env
DEEPSEEK_API_KEY=your_api_key
DEEPSEEK_MODEL=deepseek-chat
TDX_ENABLED=false
EMAIL_ENABLED=false
WEBHOOK_ENABLED=false
```

## 启动命令

```bash
streamlit run app.py --server.port 8503 --server.address 127.0.0.1
```

## 常用增强配置

### 1. 使用 TDX 实时行情

```env
TDX_ENABLED=true
TDX_BASE_URL=http://127.0.0.1:8181
```

### 2. 开启邮件通知

```env
EMAIL_ENABLED=true
SMTP_SERVER=smtp.qq.com
SMTP_PORT=587
EMAIL_FROM=your_email@qq.com
EMAIL_PASSWORD=your_authorization_code
EMAIL_TO=target@example.com
```

### 3. 开启 Webhook 通知

```env
WEBHOOK_ENABLED=true
WEBHOOK_TYPE=dingtalk
WEBHOOK_URL=https://oapi.dingtalk.com/robot/send?access_token=xxx
WEBHOOK_KEYWORD=股票
```

## 排查顺序

1. 先确认 `DEEPSEEK_API_KEY` 已配置。
2. 再确认数据源是否可用，必要时关闭 `TDX_ENABLED` 做对照。
3. 最后检查通知配置是否完整。

## 注意

系统当前只有“分析 + 提醒 + 手工登记”模式，不再支持自动下单。
